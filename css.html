The syntax of external CSS

selector{property : property value ;}
SELECTOR As in who, in whom you want changes in.
PROPERTY As in what ,what you want to change
PROPERTY VALUE As in how, how you wish to make a change in your property.

SELECTORS

CLASS SELECTORS

Use class in a tag lets say ex; body, h1,p etc..
Give the name form which you can refer to it in the external CSS file
Use ('.') in front of the class name and then you can acess it as.

ID SELECTORS

For ID selctors it is kinda similar to class selectors but in external CSS we use (#) in fornt of the ID name .

Well the only difference between them is ID selectors are unique you cannot use them again with the same reference so you would use ID to identify elements where there is only one of them on a particular page. You will get to access it only once whereas the class selectors can be addressed as many times you need.

So use classes when you want to apply the same style to a group of related items and use the ID to apply to a specific style to a single element on you web page.And these selectors overrides the tag selectors.

The new tag - "div"

A div stands for a content division element , an it basically allows you to split up or divide your content into separate containers or boxes so that you can affect the layout of each box separately.

The box model in CSS

We know that every single HTML element is treated as box by css and by styling that box, changing its height/width, margin , padding, etc. that's how we can change the layout of our website.
